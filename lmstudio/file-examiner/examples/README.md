# Example Usage Scripts for File Examiner Add-on

This directory contains example scripts showing how to use the File Examiner add-on.

## Prerequisites

Make sure the File Examiner add-on is installed and active in LM Studio before running these examples.

## Running Examples

Each script demonstrates different aspects of the add-on:

1. `basic_file_reading.js` - Simple file reading operations
2. `directory_exploration.js` - Directory browsing and navigation
3. `file_search.js` - Pattern-based file searching
4. `security_demo.js` - Demonstrates security features

To run an example:
```bash
node examples/basic_file_reading.js
```
